# 중고거래 커뮤니티 플랫폼

## 주요기능
1. 회원가입, 로그인
2. 게시글 CRUD, 이미지 첨부
3. 댓글
4. 회원정보 수정

## 개발기간
2023.03-2023.07

## 개발인원
1명(개인프로젝트)

## 개발환경
- Springboot
- java
- JPA
- javascript,AJAX
- HTML
